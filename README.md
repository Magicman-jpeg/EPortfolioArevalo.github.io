# EPortfolioArevalo

**Live Site:** `https://magicman-jpeg.github.io/EPortfolioArevalo.github.io/`

E-Portfolio of Jayclifferson P. Arevalo for the course Object-Oriented Programming.
Features
- Single-page app with hash-based routing (no server required)
- Collapsible project cards with smooth animation
- Inline PDF viewing via `<embed>` and PNG via `<img>`
- Fully responsive
- Dark theme with purple gradient accents
- Graceful fallback when PDFs/images are not yet uploaded

Built for BSIT Object-Oriented Programming
